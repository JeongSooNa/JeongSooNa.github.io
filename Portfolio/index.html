Project #2

in-silico 신약개발이란 세포 내(in-vivo), 시험관 내(in-vitro)에서 연구하는 것이 아닌
컴퓨터를 사용한 신약개발 연구로, 단백질 구조학을 기반으로 한 단백질, 화합물 3D 구조에 대해 파악하고, 결합구조를 예측하는 업무를 수행했습니다.
RCSB PDB (Protein Data Bank) 구조를 주로 사용했으며, 자체 화합물 라이브러리를 사용해 물질을 개발하는 과정으로 Public Tool과 함께 사내 자체 개발 프로그램을 활용해 단계적으로 진행했습니다.

연구소 내 운영 파트를 담당하면서 제가 맡은 업무는 다음과 같습니다.
  1. 운영 환경 세팅 및 프로그램 설치
  2. 운영에 필요한 run, 자동화 스크립트 작성 (python, bash script 활용)
  3. 자체 프로그램 운영(Operating)
  4. 결과 검증 작업 (Public Tool 사용)
  5. 데이터 핸들링 및 최종 리포트 작성

먼저, 타겟이 되는 단백질 구조를 확보한 후 Binding Pocket을 지정 해 해당 위치에 결합할 수 있는 화합물들을 대상으로 Screening을 진행합니다.
여기서 사용되는 Public Tool로는 Gnina, AutoDock 등의 Molecular Docking 프로그램이 있습니다.
이런 다양한 프로그램을 사용하면서 사용 환경 설정에 대한 지식, Github를 활용한 프로그램 세팅, Docker 환경에서의 작업 및 이미지, 컨테이너 구성 등의 스킬을 익힐 수 있었습니다.

이 외에도 다양한 Public Tool들을 세팅, 운영, 실무에 적용한 내용이 Github에 저장되어있습니다.

사내 4000여대의 CPU, GPU 서버 보유 환경에서 병렬처리 방법으로 작업 수행 경험을 할 수 있었습니다.
여기서 기초적인 Linux 환경에서의 작업, 환경 설정, bash script 등을 배울 수 있었고,
하드웨어 유지보수, 신규 서버 설치 작업 등에도 지원을 통해 참여할 수 있었습니다.
